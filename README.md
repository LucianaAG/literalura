# Literalura

Aplicación desarrollada en Java con Spring Boot que consume la API pública Gutendex para buscar libros del proyecto Gutenberg y almacenarlos en una base de datos PostgreSQL.

## Funcionalidades

- Buscar libros por título usando la API Gutendex
- Guardar libros y autores en la base de datos
- Listar libros registrados
- Listar autores registrados
- Buscar autores vivos en un año determinado
- Listar libros por idioma
- Mostrar cantidad de libros por idioma

## Tecnologías utilizadas

- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Jackson
- Gutendex API
